## üö® Common Signup Issues & Solutions

### Issue 1: "Card verification failed"

**Solutions:**
1. Ensure card has online/international transactions enabled
2. Try a different card
3. Check with your bank if they blocked Oracle
4. Some banks send OTP - check SMS
5. Wait 1 hour and try again (too many attempts can cause temp block)

### Issue 2: "Region not available"

**Solutions:**
1. Use VPN if website shows wrong region options
2. Clear browser cache and cookies
3. Try different browser (Chrome works best)
4. Contact Oracle support via chat

### Issue 3: "Email verification not received"

**Solutions:**
1. Check spam/promotions/junk folder
2. Wait 10 minutes (sometimes delayed)
3. Click "Resend verification email"
4. Try a different email address
5. Check if email provider is blocking Oracle

### Issue 4: "Phone verification failed"

**Solutions:**
1. Ensure format is: +91 [10 digits] (no spaces)
2. Check SMS permissions aren't blocking Oracle
3. Try "Resend OTP" after 2 minutes
4. Use landline number if mobile not working (rare)

### Issue 5: "Temporary authorization charge showing"

**This is NORMAL:**
- ‚Çπ1-2 authorization hold
- Will reverse in 3-7 business days
- Not an actual charge
- Some banks show it, some don't

### Issue 6: "Account pending verification"

**What to do:**
1. Check email for "Action Required" from Oracle
2. May need to submit ID proof (rare for India)
3. Usually resolves in 24-48 hours
4. Contact support via: https://www.oracle.com/corporate/contact/
