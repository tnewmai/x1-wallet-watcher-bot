## üéØ WHAT IS A RUG PULL?

A **rug pull** is a crypto scam where:

1. **Token Creation:** Scammer creates a new token
2. **Liquidity Addition:** Adds liquidity to a DEX (e.g., Token/X1 pair)
3. **Marketing/Hype:** Promotes the token to attract buyers
4. **Liquidity Removal:** Removes all liquidity, leaving buyers with worthless tokens
5. **Exit:** Scammer disappears with the funds

### Detection Methods Used

Our bot detects rug pullers by analyzing:

- **Transaction History:** Looking for LP removal patterns
- **Deployer Behavior:** Checking if wallet has rugged previous tokens
- **Funding Chain Analysis:** Tracing wallet creation and funding sources
- **New Wallet Detection:** Identifying freshly created deployer wallets
- **Serial Deployer Patterns:** Finding deployers with multiple tokens
